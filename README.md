# ISW
Proyecto final del curso de Ingeniería de Software ESCOM 2023

## Ejecución Proyecto

### Backend

Solicitar archivo `.env` y guardarlo dentro del folder `api`, debemos cambiar al directorio `api` y ejecutar

```console
$ npm i
$ npm run dev
```
